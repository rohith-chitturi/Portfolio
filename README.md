## ⚡ Quick Start / Local Deployment

Because this portal utilizes modern vanilla web components, it is entirely serverless and can be run without compiling:

```bash
# Clone the repository
git clone https://github.com/rohith-chitturi/<your-repo-name>.git
cd <your-repo-name>

# Option A: Start a quick Python local server
python -m http.server 8000

# Option B: Run via Node's serve utility
npx serve .
```
Open `http://localhost:8000` (or the local port provided) in your browser.
